使用hexo搭建博客详细步骤

1 前言

1.1 不同博客搭建方式的对比

可选方案
1、Git+Github+Markdown+jekyll （免费）
2、Git+Github+Markdown+hexo （免费）
3、虚拟主机＋插件＋Wordpress （付费）

放弃jekyll原因
–初始效果不明显
–不需要复杂需求
–需要ruby语言基础

放弃wordpress原因
–付费
–需要php语言基础

1.2 hexo搭建步骤

使用Hexo+Github一步步搭建属于自己的博客（基础）

a) 准备好node和git环境
node、npm、修改npm仓库地址、git
b) github创建项目，通过Github Pages选择Themes后，可以通过网址访问github项目
c) 安装hexo
$npm install hexo -g 全局安装Hexo
$hexo -v，检查hexo是否安装成功
d) 初始化hexo项目
进入项目的空文件夹
$hexo init
$npm install
$hexo g 生成项目文件
$hexo s 开启服务
e) 修改hexo配置文件，保存github仓库地址
f) 创建新博客
$hexo new post “博客名”
g) 安装扩展
$npm install hexo-deployer-git –save
h) 发布到github
$hexo d -g

1.3 遇到的坑

a) 无法加载css文件
修改_config.yml的url和root值
b) 使用hexo-admin插件
$npm install –save hexo-admin
$hexo server -d
